My Research

CLIP Implementation from Scratch

An implementation of the CLIP model trained on the Flickr30k dataset. As part of a collaborative project for the FML (CS725) course at IIT Bombay, the model learns cross-modal relationships to generate images from captions and vice-versa.

Robust 2D Direct Preference Optimization

This work explores the advantages of 2D-DPO over traditional DPO for aligning LLMs with human preferences. It addresses the model's sensitivity to label noise and proposes a robust version backed by analysis and empirical results.

Efficient Data Selection in Collaborative ML

This research enhances the Mycroft framework, which uses subset selection algorithms like GradMatch for efficient model training. Key contributions include codebase refactoring for modularity, integrating Vision Transformers (ViTs), and fixing training inconsistencies.
